# 架构总览

全部组件通过 Docker Compose 编排，一键部署。

## 数据流

### 导入阶段

```
原始数据 → 格式检测 → 对应解析器 → Chunk 切分 → BGE-M3 Embedding → Qdrant 存储
                                        ↓
                                   元数据提取（文件路径、语言、函数签名等）
```

### 查询阶段

```
用户查询 → Query 改写/扩展
              ↓
         BGE-M3 Embedding
              ↓
         Qdrant 向量检索 + 元数据过滤
              ↓
         BGE-Reranker 精排
              ↓
         Top-K Chunk + 元数据 → Prompt 组装
              ↓
         vLLM (Qwen2.5-Coder-32B) 生成回答
              ↓
         流式输出（SSE）→ 前端渲染
```

## 设计原则

- **渐进式复杂度**：从线性 RAG 开始，按需引入多路检索、Agentic RAG
- **模型可插拔**：LLM、Embedding、Reranker 均可替换
- **数据源可扩展**：通过解析器接口扩展新的数据类型
- **单机优先**：不引入不必要的分布式复杂度
